A full-stack webshop built as part of a university course at the IT University of Copenhagen. The project focuses on creating a responsive and user-friendly e-commerce platform with a modern tech stack and clean UI.

Key Features (planned and in progress)

Product listing and single product views

Shopping cart functionality

User authentication and profile management

Responsive design

Admin dashboard for managing products

Status
Currently in development. Backend implementation is in progress. UI is being refined for responsiveness and usability.
